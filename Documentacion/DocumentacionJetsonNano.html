Nvidia Jetson Orin Nano

Guía de Configuración e IA Local

Documentación detallada del proceso de puesta en marcha de la Nvidia Jetson Orin Nano, incluyendo montaje físico, configuración inicial del sistema operativo, preparación del almacenamiento SSD y despliegue de modelos de lenguaje.

1. Montaje y Hardware

1.1. Tarjeta SD y Alimentación

Insertamos la tarjeta microSD en la ranura ubicada en la parte inferior de la placa, bajo el disipador. Es crucial utilizar el cable de alimentación con conector europeo suministrado, ya que el original es incompatible.

1.2. Conexión al monitor

Conectamos el sistema a traves de display port al monitor.

2. Configuración Inicial (Ubuntu)

2.1. Licencias e Idioma

Aceptamos los términos de licencia de Nvidia. Seleccionamos "Español" como idioma del sistema y configuramos la distribución del teclado correspondiente.

2.2. Red y Zona Horaria

Nos conectamos a la red Wi-Fi disponible para descargar actualizaciones. Configuramos la zona horaria en "Madrid".

2.3. Usuario y Particionamiento

Creamos el usuario administrador. En la configuración de particiones (APP Partition Size), asignamos el tamaño máximo posible para aprovechar todo el espacio.

2.4. Navegador y Actualización

Aceptamos la instalación del navegador Chromium. Al finalizar la configuración inicial y llegar al escritorio, ejecutamos el "Software Updater" para actualizar el sistema a la última versión disponible y reiniciamos.

3. Configuración SSD (Terminal)

Para alojar los modelos de IA sin llenar la memoria principal, configuramos un disco SSD NVMe secundario mediante la terminal.

3.1. Detección y Formateo

Comprobamos que el dispositivo está conectado (`lspci`) y lo formateamos en formato ext4.

sudo mkfs.ext4 /dev/nvme0n1

3.2. Montaje y Persistencia

Creamos el directorio `/ssd`, montamos el disco y editamos el fichero `fstab` para asegurar que se monte automáticamente al iniciar el sistema.

sudo mkdir /ssd sudo mount /dev/nvme0n1 /ssd sudo nano /etc/fstab

3.3. Permisos

Cambiamos los permisos de la carpeta para que el usuario actual tenga control total sobre ella (Mas el historial completo de comandos).

sudo chown ${USER}:${USER} /ssd

4. Instalación IA (LM Studio)

4.1. Instalación de Dependencias

Descargamos el archivo AppImage de LM Studio. Para ejecutarlo en Ubuntu 22.04, es necesario instalar previamente la librería `fuse3`.

sudo apt install fuse3 chmod +x LM-Studio-0.3.32-2-arm64.AppImage

4.2. Selección y Descarga de Modelos

Abrimos LM Studio y navegamos por la biblioteca de modelos. Podemos ver una lista de diferentes modelos disponibles. Seleccionamos un modelo eficiente para nuestro hardware, como Qwen 4B.

Accedemos a los detalles del modelo seleccionado para verificar la compatibilidad y procedemos a la descarga.

4.3. Instalación Completada y Carga

Una vez pulsado el botón de descarga, esperamos a que finalice el proceso. El sistema nos notificará con el mensaje "Download Completed", momento en el cual el modelo ya está instalado localmente en nuestro SSD y listo para ser cargado en memoria ("Load Model").

4.4. Pruebas de Inferencia (Chat)

Con el modelo cargado, iniciamos una conversación. Probamos la capacidad de razonamiento de la IA con preguntas lógicas (por ejemplo, sobre la palabra "Ornitorrinco"), confirmando que la Jetson Nano está ejecutando el modelo correctamente.
